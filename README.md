# NedCertify Simple Backend

Backend de démonstration simple pour NedCertify, compatible avec Render.

## Comptes de démo
- admin@nedcertify.com / admin123
- ministry@nedcertify.com / ministry123
- auditor@nedcertify.com / auditor123

## Déploiement Render
- Build Command: npm install
- Start Command: npm start

## Variables
- NODE_ENV=production
- PORT=10000
- JWT_SECRET=une-valeur-forte
- CORS_ORIGIN=https://nedcertify.com
- PUBLIC_APP_URL=https://nedcertify.com
